# Mini CRM (Vanilla JS)

Мини-CRM для учёта клиентов: добавление, удаление, изменение статуса, сохранение данных в браузере.

## Features
- Add client
- Delete client (with confirmation)
- Toggle client status (active/inactive)
- Persistent storage via localStorage
- Responsive UI layout (basic)

## Tech stack
- HTML
- CSS
- JavaScript (Vanilla)
- localStorage

## How to run
Open `index.html` in a browser.
